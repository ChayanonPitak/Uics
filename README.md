# Requirements
- CMake version > 3.19 (https://cmake.org/install/)
- sw (software network) version > 1.0.0 (https://software-network.org/)

1. mkdir build && cd build
2. cmake ..
3. cmake --build .
4. exe file shoud be in ./build/Debug
